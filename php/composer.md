# composer

Runs a security audit of your dependencies and reports known vulnerabilities.
```bash
composer audit
```

Updates dependencies according to `composer.json` and writes new versions to `composer.lock`.
```bash
composer update
```

Lists installed packages with available updates.
```bash
composer outdated
```

Runs Composer's self-update to update the Composer binary to the latest stable version.
```bash
composer self-update
```
