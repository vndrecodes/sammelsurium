# Directories

```shell
/bin        # Systemprograms
/dev        # Devices
/etc        # Configs
/root       # root dir
/home       # user dir
/lib        # shared libs (dynamically linked os programs)

/mnt        # mount pts
/media

/usr        # unix system resources (appliactions, manpages) for a user
/usr/bin
/usr/lib
...

/opt        # Mostly proprietary programs
/proc       # Processes
/sbin       # Systemprograms which needs special permissions
```
