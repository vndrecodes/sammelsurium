# npm useful commands

```shell
$ npm outdated                 # show outdated packages

$ npm update                   # update packages
$ npm update --save            # add to packages.js (no longer needed)
$ npm update --save-dev        # add to packages.js (devDependencies)
$ npm update --save-optional   # add to packages.js (optionalDependencies)

$ npm list
```
