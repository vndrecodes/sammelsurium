# General
```
cmd + shift + <         Switch between windows from same app  
cmd + `                 Switch between windows from same app  
cmd + opt + M           Minimize all windows from same application
F5 or opt + esc         Speech
```

# Safari
```
cmd + shift + D         Zur Leseliste hinzufuegen  
crtl + ,                Passwoerter oeffnen  
```

# Finder
```
option + left/right     collapse/expand folders  
cmd + A -> left/right   collapse/expand all folders
cmd + opt + W           close all windows
cmd + shift + .         show hidden files
```
